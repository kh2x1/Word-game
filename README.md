# 🧠 Memory Quest — رحلة الذاكرة (PWA)

لعبة ذاكرة احترافية بأسلوب Neon — قابلة للتثبيت كتطبيق!

-----

## 📦 محتويات الحزمة

```
memory-quest-pwa/
├── index.html              ← اللعبة الكاملة
├── manifest.json           ← إعدادات التطبيق
├── service-worker.js       ← يخلي اللعبة تشتغل بدون نت
├── icon-192.png            ← أيقونة 192×192
├── icon-512.png            ← أيقونة 512×512
├── icon-maskable.png       ← أيقونة قابلة للقص (Android)
├── apple-touch-icon.png    ← أيقونة iOS
└── favicon-32.png          ← أيقونة المتصفح
```

-----

## 🚀 طريقة النشر (3 خطوات)

### الخطوة ١ — حمّل المجلد كامل

لازم ترفع **كل الملفات** مع بعض، مش بس HTML.

### الخطوة ٢ — ارفعها على Netlify Drop (الأسهل)

1. روح [app.netlify.com/drop](https://app.netlify.com/drop)
1. اسحب **مجلد `memory-quest-pwa` كامل** على الصفحة
1. ينعطيك رابط مثل `https://amazing-name.netlify.app`

### الخطوة ٣ — شارك الرابط!

أي شخص يفتح الرابط، يقدر:

- 📱 **على الجوال:** يظهر له بانر “ثبّت اللعبة على جوالك”
- 💻 **على الحاسوب:** يظهر زر التثبيت في شريط العنوان

-----

## 📲 كيف تثبّت اللعبة على الجوال

### Android (Chrome / Samsung Internet)

1. افتح الرابط
1. سيظهر بانر “ثبّت اللعبة على جوالك” — اضغط **تثبيت**
1. أو من قائمة المتصفح ⋮ → “إضافة للشاشة الرئيسية”

### iPhone / iPad (Safari فقط)

1. افتح الرابط في **Safari** (ما يشتغل في Chrome على iOS)
1. اضغط زر المشاركة ⬆️ في الأسفل
1. اختر “إضافة إلى الشاشة الرئيسية”
1. اضغط “إضافة”

-----

## ✨ المميزات

✅ **يشتغل بدون نت** — بعد أول فتح
✅ **سريع جداً** — يفتح فورياً
✅ **يحفظ تقدمك محلياً** — حتى لو طفيت الجوال
✅ **يبان كتطبيق حقيقي** — بدون شريط متصفح
✅ **بانر تثبيت تلقائي** — يظهر بعد 4 ثواني
✅ **هوية بصرية احترافية** — أيقونة Neon

## 🛠️ تشغيلها محلياً للتطوير

⚠️ **مهم:** PWA ما تشتغل بفتح `index.html` مباشرة — لازم سيرفر.

### الأسهل (Python):

```bash
cd memory-quest-pwa
python3 -m http.server 8000
```

افتح `http://localhost:8000` في المتصفح.

### Node.js:

```bash
npx serve .
```

## ❓ مشاكل شائعة

**البانر ما يظهر؟**

- تأكد إنك ترفع كل الملفات مع بعض
- جرب من متصفح مختلف
- امسح ذاكرة التخزين المؤقت

**ما يثبّت على iOS؟**

- لازم تفتحه من Safari مش Chrome
- iOS ما يدعم زر التثبيت التلقائي

**يشتغل بدون نت؟**

- لازم تفتح الرابط أول مرة وأنت متصل
- بعدها سيرفس وركر يخزن كل شيء
